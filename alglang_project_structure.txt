================================================================================
  KEY DESIGN NOTES
================================================================================

  1. src/core/       — Zero dependencies on runtime/ or builtins/.
                       Can be tested in isolation.

  2. src/runtime/    — Depends on core/ for types only.
                       The GC knows nothing about the VM internals.

  3. src/builtins/   — Depends on runtime/ and diagnostics/ only.
                       Each builtin_X.c is a standalone translation unit.

  4. src/graphics/   — Compiled only when BUILD_GRAPHICS=ON.
                       No other module depends on graphics/.

  5. src/i18n/       — Zero dependencies. Loaded first. Used by all modules.

  6. src/diagnostics/ — Zero dependencies. Used by all modules.

  7. tests/          — Links against the alglang_core static library.
                       Each test_X.c maps 1:1 to a source module.

  8. Arena allocator — AST nodes are allocated from an arena (arena.h).
                       The entire AST is freed in one arena_free() call
                       after compilation, before the VM starts.

================================================================================
  BUILD ARTIFACTS
